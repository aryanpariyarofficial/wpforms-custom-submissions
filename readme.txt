A custom plugin to handle WPForms submissions and display them in the WordPress admin area.

== Description ==

WPForms Custom Submissions is a plugin designed to extend the functionality of the WPForms plugin by saving form submissions to a custom database table and displaying them in a custom admin page. This plugin provides an easy way to manage and review form submissions directly from your WordPress dashboard.

**Features:**
* Save WPForms submissions to a custom database table.
* Display submissions in a custom admin page.
* Search, sort, and paginate submissions.
* View detailed submission data in a popup.
* Delete submissions from the admin page.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wpforms-custom-submissions` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the WPForms Submissions menu item in the admin sidebar to view and manage form submissions.

== Frequently Asked Questions ==

= Does this plugin work with the free version of WPForms? =

Yes, this plugin works with both the free and premium versions of WPForms.

= Can I export submissions? =

Currently, this plugin does not support exporting submissions, but this feature may be added in future updates.

= How can I search through the submissions? =

You can use the search box on the WPForms Submissions admin page to search through submissions by any field.

= Is there pagination for the submissions? =

Yes, the submissions are paginated to ensure that the admin page remains easy to navigate, even with a large number of submissions.
